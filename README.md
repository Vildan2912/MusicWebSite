ОБЩАЯ ИНФОРМАЦИЯ О ПРОЕКТЕ (по ходу действий будет изменяться)

Название проекта:
Музыкальный стриминговый сервис "MyzON"

Краткое описание:
Разработка веб-сайта для прослушивания музыки с поддержкой различных ролей пользователей (гость, слушатель, артист, администратор), системой рекомендаций, а также расширенными возможностями для премиум-пользователей (прослушивание без рекламы, высокое качество звука, возможность офлайн прослушивания через скачивание треков).

Цель проекта:
Создать полнофункциональный музыкальный стриминговый сервис, обеспечивающий:

1) для слушателей — удобный доступ к музыкальному контенту и персонализированные подборки;

2) для артистов — инструменты для публикации треков, просмотра статистики и взаимодействия с аудиторией;

3) для администраторов — систему модерации контента и управления пользователями.

Технологический стек:

Frontend: HTML, CSS, JavaScript (возможно React/Vue для упрощения)

Backend: Python (Django/FastAPI/Flask)

База данных: PostgreSQL / MySQL

Хранение файлов: локальное или облачное хранилище (для аудиофайлов)

Версионный контроль: Git (GitHub)

Основные модули системы:

1) Модуль аутентификации и управления пользователями (регистрация, вход, роли)

2) Модуль каталога музыки (треки, альбомы, артисты, поиск)

3) Модуль плеера (потоковое воспроизведение, превью, реклама)

4) Модуль подписок (премиум-доступ, офлайн-скачивание)

5) Модуль плейлистов (создание, редактирование, удаление)

6) Кабинет слушателя (настройки, история прослушиваний, любимые треки)

7) Кабинет артиста (загрузка треков, статистика, доходы)

8) Панель администратора (модерация, управление пользователями, реклама)

10) Система рекомендаций (персонализированные подборки)
