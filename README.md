ОБЩАЯ ИНФОРМАЦИЯ О ПРОЕКТЕ (по ходу действий будет изменяться)
Название проекта:
Музыкальный стриминговый сервис "MyzON"

Краткое описание:
Разработка веб-сайта для прослушивания музыки с поддержкой различных ролей пользователей (гость, слушатель, артист, администратор), системой рекомендаций, а также расширенными возможностями для премиум-пользователей (прослушивание без рекламы, высокое качество звука, возможность офлайн прослушивания через скачивание треков).

Цель проекта:
Создать полнофункциональный музыкальный стриминговый сервис, обеспечивающий:
для слушателей — удобный доступ к музыкальному контенту и персонализированные подборки;
для артистов — инструменты для публикации треков, просмотра статистики и взаимодействия с аудиторией;
для администраторов — систему модерации контента и управления пользователями.

Технологический стек:
Frontend: HTML, CSS, JavaScript (возможно React/Vue для упрощения)
Backend: Python (Django/FastAPI/Flask)
База данных: PostgreSQL / MySQL
Хранение файлов: локальное или облачное хранилище (для аудиофайлов)
Версионный контроль: Git (GitHub)

Основные модули системы:
Модуль аутентификации и управления пользователями (регистрация, вход, роли)
Модуль каталога музыки (треки, альбомы, артисты, поиск)
Модуль плеера (потоковое воспроизведение, превью, реклама)
Модуль подписок (премиум-доступ, офлайн-скачивание)
Модуль плейлистов (создание, редактирование, удаление)
Кабинет слушателя (настройки, история прослушиваний, любимые треки)
Кабинет артиста (загрузка треков, статистика, доходы)
Панель администратора (модерация, управление пользователями, реклама)
Система рекомендаций (персонализированные подборки)
